Variables
=========
In javascript variable creation has two parts
1. variable declaration : 
    var a;
2. variable definition
    a = 10;

Note: declaration and definition can be written in single line
    var a = 10; 

Declaration of variable in javascript with 3 key words
1. var
2. let
3. const

declaration
    var a;  ==> it is old style and scope is global
    let a; ==> new style and scope is block level
    const A; ==> It is constant which means variable will not accept any kind of assignment that means once variable is created it will not be
    modified.


Example
======
    var a = 100;
    {
        console.log(a);  // 100
    }
    console.log(a); // 100

    var a = 111;
    {
        var a = 222;
        console.log(a);  // 222
    }
    console.log(a);   // 222

    var a = 333;
    {
        let a = 444;
        console.log(a);    // 444
    }
    console.log(a);   // 333

let a = 222;
{
    console.log(a);  // 222
}
console.log(a);  // 222


let a = 333;
{
    let a = 444;
    console.log(a);   /// 444
}
console.log(a);  // 333


var a = 333;
{
    let a = 444;
    console.log(a);   /// 444
}

logical Operators
=================
and ==> &&
or  ==> ||

true && true ==> true
true && false ==> false
false && true ==> false
false && false ==> false

true || true ==> true
true || false ==> true
false || true ==> true
false || false ==> false

console.log(5>6 || 7<8); true
console.log(5>6 || 7>8); false

console.log(0 && 4); 0

console.log(b); //==> 5

console.log(b);

var a = false;
